Descripción de la librería

- Matplotlib es una librería del lenguaje de programación Python utilizada para la visualización de datos.
Permite crear gráficos de líneas, barras, pastel, dispersión, histogramas y muchos otros tipos de visualizaciones de manera sencilla y profesional.
Es ampliamente usada en educación, ciencia, ingeniería y negocios, ya que transforma los datos numéricos en representaciones gráficas claras y comprensibles.

Gracias a su facilidad de uso y su compatibilidad con otras librerías como NumPy y Pandas, Matplotlib es una de las herramientas más importantes para el análisis y la presentación de resultados dentro del ecosistema de Python.

- Instrucciones de instalación
  
Antes de usar Matplotlib, asegúrate de tener Python 3.8 o superior instalado en tu equipo.
Luego, abre la terminal o consola y escribe el siguiente comando para instalar la librería:

pip install matplotlib

Consejo: si usas un entorno virtual (venv), actívalo antes de ejecutar el comando para mantener tu proyecto organizado.
